checkCharacterCoinCollisions(world)

Checks for collisions between the character and coins in the game world. If a collision is detected and the character's coins are not at their maximum, the character's coins are increased, the coin bar status is updated, and the collided coin is removed from the level.
Parameters:
Name Type Description
world object The game world object containing level and character information.

checkThrowableObjecGround(world)

Checks for collisions between throwable objects and the ground in the game world. If a throwable object's y-coordinate is greater than or equal to 360 (indicating it's on or below the ground) and the object has not already hit, the object is marked as already hit, collision-related properties are updated, sound effects are played, and the object is removed.
Parameters:
Name Type Description
world object The game world object containing throwable objects information.

checkThrowableObjectEndbossCollisions(world)

Checks for collisions between throwable objects and the end boss in the game world. If a collision is detected and the throwable object has not already hit the boss, the throwable object is marked as already hit, the boss's damage from collision is calculated, collision-related properties are updated, sounds are played, and the throwable object is removed.
